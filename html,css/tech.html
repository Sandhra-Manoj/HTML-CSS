HTML

The HyperText Markup Language or HTML is the standard markup language for documents designed to be displayed in a web browser.

It defines the meaning and structure of web content.

It is often assisted by technologies such as Cascading Style Sheets and scripting languages such as JavaScript.

JAVA

Java is a high-level, class-based, object-oriented programming language that is designed to have as few implementation dependencies as possible.

ava was originally developed by James Gosling at Sun Microsystems. It was released in May 1995 as a core component of Sun Microsystems' Java platform. The original and reference implementation Java compilers, virtual machines, and class libraries were originally released by Sun under proprietary licenses.
